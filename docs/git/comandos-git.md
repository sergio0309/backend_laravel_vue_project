# Comandos de GTI (Iniciales)
## Requerimientos
- Descargar GIT
- Crear una cuenta en GITHUB
## Configurar GIT
- Ejecutar los siguientes comandos
```
git config --global user.name "Nombre"
git config --global user.email "su.correo@gmail.com"
```
